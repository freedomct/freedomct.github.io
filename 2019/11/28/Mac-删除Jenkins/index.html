Mac 删除Jenkins

背景介绍:安装了Jenkins 但是在使用中配置出错了，所以想卸载重新安装。但是在安装过程中，却发现即使安装后，前期的配置还是存在。但是明明通过命令确定卸载啦。通过brew install Jenkins 也就通过brew uninstall Jenkins。uninstall 后 配置还会保留在本地，如图所示，所以如果想完全删除Jenkins 要通过2个步骤: 第一卸载，第二 删除本地配置，这样重新安装才会从头开始 而不是把以前的错误配置也保留在最新安装的版本中。
