四月 18, 2019

#个人目前正在使用的 Google 插件推荐

目前正在使用的

  • OneTab

当浏览器中打开的窗口太多时，就会使用过多内存但是有些界面的内容有没有阅读想推迟使用这个就可以帮到用户保存当前打开的窗口节约内存了。
虽然Google 后期也更新了稍后阅读的功能，但是可能是已经习惯了 所以就没有更换。
优点: 便捷、易导出导入
缺点: 无法离线同步 只能手动本地保存

  • Application Launcher for Drive (by Google)

Google 云盘 (由于连接Google 服务器所以你懂的)

  • AHA Music - Music Identifier

捕捉网页中的流媒体 当你打开一个网页时 突然听到一首悦耳的旋律当你想保存的时候就用到这个插件了(个人很少用到 现在各个音乐播放器都有听歌识曲功能 这个也该抛弃了)

  • Axure RP Extension for Chrome

产品必备 但是有些产品是不会用Axure 的 所以看个人了

  • baidu-dl

百度网盘开会员了 没有那就只能忍受10k的下载速度了 虽然有试用会员 但是也就是一分钟的事情 还要看网速 否则那点会员时间根本就可以忽略不计 那么这个就可以解析出高速下载地址给你啦 只要你的带宽够强 (慎用，由于利益有关，百度一直在封杀这类)

  • Bookmarks clean up

书签保存的很多嘛 好久都么用了 以至于有些书签都无法访问了 这个就是帮你清理书签的工具 一共有四个功能:查找重复书签、清理空文件夹、合并重复文件夹、筛选出书签的可访问状态

  • Clearly

打开一个网页 一堆广告 那么就用这个清理下吧 和safari 阅读模式类型的一个插件 同类型的 还有简悦 看个人选择

  • cross-request

YApi 跨域请求 主要是web端的 如果 是iOS 中嵌入web界面 也可以使用

  • Dark Reader

Google 推出了暗黑系列 但是苦于穷逼一个 所以只能在强大的网络中寻找替代品了 这个就是一个无法媲美Google 暗黑的一个替代品 但是可以自定义在 界面上显示如果要求不是很高的话 可以使用一下 不过以后Google 应该也会开放暗黑系列吧 毕竟现在各大厂商都在添加这个功能

  • Darkness - Beautiful Dark Themes

这个就是原生的暗黑系列 不过 要收费的 土豪随意

  • DuckDuckGo Privacy Essentials

还在网络上裸奔嘛 起码弄块遮羞布 不要全裸吧 鸭子就是为了保护你的隐私的一个搜索引擎 虽然还没普及 但是还是在逐步强大起来(自备梯子)

  • ImageAssistant Batch Image Downloader

花瓣有段时间挂了 然后全时间找能抓取网上也所有图片并能打包下载的功能 也就这么火了 平时我们也可以用到 当看到自己喜欢的也可以一键下载啦

  • Google Docs Offline

words office 是不是还在用盗版的 个人使用还可以 但是如果是公司使用的话 那到时候可能就要收到船票啦 Google 离线文档 了解下 免费试用 (自备梯子)

  • EditThisCookie

当访问网站时 有些个人隐私信息不想被共享但是浏览器的cookie 设置又有点繁琐那么这个插件就可以直接编辑cookie 可以修改 删除 新增

  • Instapaper

稍后阅读 当时打开一个学习网页正要学习 时 居然下班了 着急回家的心情是无法言喻的 又不想复制粘贴 发送到微信啥的 那么直接点击一下 稍后阅读 自动同步 然后打开客户端 就直接同步阅读啦 另 Google 也添加了稍后阅读功能啦

  • JSONView

当和服务器交互时 一堆数据回来 看着就像一坨xx 那么jsonview 就会把返回的数据进行一个格式化 让数据看起来很舒服

  • Isometric Contributions

作为一个互联网工作者 GitHub 这类代码管理 肯定是要接触的 每天 或者每年 一段时间 你的工作效率是这样的 那么Isometric 就会把你每天在GitHub 上的贡献 用柱状体清晰的显示出来 这样就可以直观明了的看到自己的辛勤劳动成果了

  • Local CDN

优化 本地CDN

  • Octotree

查看一个GitHub 项目 想要大概了解整体的一个结构 那么Octotree就会为你生成一个树状结构共你查看

  • Open In IINA

IINA 播放器的插件

  • Play HLS M3u8

直播流

  • Reggy

打开一个网址 居然需要注册登陆才能查看或者下载 这就操蛋了 但是又不想泄露自己的个人邮箱 只是临时使用一下 Reggy就可以帮你产生一个临时数据 供你使用

  • SimpRead - Reader View

阅读工具 简化阅读

  • Responsive Web Design Tester

当前大多数都是移动端时代 PC份额很少了 所以当我们做项目时 就要时时适配移动端 当前就为你提供 各种终端机模拟

  • Tampermonkey BETA

油猴 你懂的

  • Wappalyzer

  • Wikiwand: Wikipedia Modernized

维基百科

  • 购物党自动比价工具

到了双十一 商家优惠啦 其实都是假象 那么我们怎么知道了 通过自动比价 查看历史数据 就知道当前商家 是否抬价后进行打折了 也可以设置降价提醒 直接目前电商平台的各大商家 只能在 web端使用
